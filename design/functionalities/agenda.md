# Agenda

Given two tables, and generate a agenda for a meeting.

Inputs:
- A table of sessions, durations and the associated roles.
- A table of role takers, with required information for some roles.


Outputs:
- A agenda in html with predefined styles.
- Version for viewing: plain html with simple style to be friendly to mobile devices.
- Version for printing: One two-sided pager, with rich information of the club.


The page in draft by default and can be viewed and edited by admins. It's published when ready. After published, it can only be editted by admins.
